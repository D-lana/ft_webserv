# ft_webserv
This project is about writing your own HTTP server.

## How to compile
+ Open folder `ft_webserv`
+ To compile:
  + `make`
+ To remove objects:
  + `make clean`
+ To remove objects and binary file (program):
  + `make fclean`
+ To re-compile:
  + `make re`
+ Execute:
  + `./webserv [config file]`
  
  Example: `./webserv test.config`
